# chengmin0266.github.io

Cheng Min（伦敦 Game Designer）个人作品集网站源码。

线上地址：https://chengmin0266.github.io

## 本地预览
直接双击 `index.html` 用浏览器打开即可。无构建工具，纯 HTML / CSS / JS。

## 文件说明
- `index.html` — 首页：intro 开场动画 + Hero + Selected Games 横向轮播 + All Projects 网格
- `personal.html` — 个人页：画作 + 小说
- `about.html` — 关于页：经历 / 工具 / 联系
- `style.css` — 所有样式（CSS 变量在文件顶部，改色调改这里）
- `lang.js` — 中英切换（HTML 元素用 `data-en` / `data-zh` 双属性）
- `images/` — 图片资源（命名规则见 `images/README.md`）

## 部署
GitHub Pages 自动部署，push 到 `main` 后 1–5 分钟生效。
